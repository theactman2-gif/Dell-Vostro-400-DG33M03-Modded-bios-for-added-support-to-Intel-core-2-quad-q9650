Modded bios to get the q9650 of the DG33M03 Motherboard variants of dell vostro 400 to recognize and use a Q9650 properly
(Maybe other motherboard variant can take this bios, Unable to test.)



Requirement : A dell vostro 400 (Or maybe other works like the inspiron 530? Dunno, Don't have one.)
A free and usable usb drive (Not a too big one! The bios won't detect it and crash when trying to go in the boot menu, 32gb is probably the max)
Rufus
And the files at hands.




How to install :

1 : Download the files i gave away called bios shit. (Obviously)

2 : Download rufus if you don't have the program on your pc.

3 : Boot rufus and you should be met with this : <img width="1920" height="1080" alt="Capture" src="https://github.com/user-attachments/assets/1d7b75c4-5d01-47fe-abd5-bdb4a4650ca3" />

4 : Select your desired devices.

5 : In boot selection, select the free-dos option <img width="1920" height="1079" alt="Capture" src="https://github.com/user-attachments/assets/669ec3b6-3143-4d15-a389-5ee956468b65" />

6 : Now press start and the formatting process and all should starts.

7 : After waiting for it to be done, Go to the root of your now free-dos usb install, Take the Bios shit file and place it at the root of the usb drive.

8 : Unplug the usb drive and go over to your dell vostro or whatever.

9 : Boot from your usb key into free-dos.

10 : Now you will be faced with a fossil like ancient dos prompt. (Can't show images of it, Would be too jank to have photos of my room)

11 : Type the command dir, after that, Find your bios shit file which should have a ~1 and two ss. (Don't ask me why, i dunno.)

12 : Once you've found said file Type in cd : "Biossh~1 or something". Then type the following command AWD899.EXE 713D1L50.BIN /py /sn /cc /cd /cp

13 : A window should appear and it should normally erase the previous bios image and overwrite it with the modified one, The computer will beep alot, It's normal, Don't worry.

14 : After it is done, It'll tell you to press f1 to finish, Press it. Then it should reboot into the bios, From there you might wanna check the setup bios page by pressing f2.

15 : The q9650 should show up and it should work fine
